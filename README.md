### A Wwise Soundengine Integration for Vektoria 3D Engine at University of Applied Sciences Kempten (Bachelor thesis) - C++:
The Vektoria game Engine which is currently mostly used by Students or Former Students of University of Applied Sciences Kempten is a modular 3D Game Engine which teaches the Game Engineering Students at University how to handle a Scenegraph like Game Engine. The Engine is actively developed by Bachelor- and Master thesis Students.

I decided to write a Sound Engine Integration because of the amount of problems many students had when it was their turn to integrate soundeffects and interactive music into games developed using Vektoria.
During development i noticed that Vektoria is missing a LowLevelIO which is necessary for Wwise Sound Engine to function.
Since There was none in existence within the Game Engine i decided to write my own and integrate it as an open source library so i am not restricted to c++14 but can use newer features from i.e. c++17 and c++20 for example to handle multiplatform filepaths without altering the original c++ version used by the engine.
